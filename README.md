# 烁公益站小游戏，囤囤鼠打野路径规划面板

> 简称：**烁站打野面板**

用于“烁公益站小游戏 - 囤囤鼠”的本地探测与打野路线规划面板。

原点附近通常资源丰富，但也是在线玩家集中活动的区域。对于追求稳定收益的玩家，可以规划前往外围的打野路线：先探测各坐标团的 Drop，再按体力选择路线，沿路收割离线玩家的 Drop。本面板负责把探测、路线规划、风险提示、体力消耗和收益计算可视化。

> 本项目仅在本机浏览器中显示游戏快照数据；请遵守游戏规则、服务条款和其他玩家的隐私边界。

## 功能

- 双快照读取玩家数据，按 STA 状态判断在线/静止，并显示在线玩家的移动方向。
- 将相邻静止玩家聚成“团”，显示团坐标、成员和 Drop。
- 自动规划最多三条互不重叠的外围打野路线，计算体力消耗与预估收益。
- 路线收益按 `（主 Drop + 附加 Drop）/2` 计算；路径范围内的额外静止团会计入附加 Drop。
- 自动避开原点活动区和在线玩家附近的危险区。
- 若以原点范围内的自己为起点，允许路线第一段安全离开原点；仅在这种情况下提示路径附近的原点在线威胁与原点内 `Drop > 15` 的离线玩家。
- 地图提供“贫瘠团”图层：淡灰色、无名称和 Drop 标签，仅用于观察低资源团形成的分布与外围人流。
- 支持自定义起点、自定义路径、地图缩放/拖动、显示筛选与点击详情。
- 参数会保存到浏览器本地存储，下次打开面板时自动恢复。

## 运行环境

- Windows
- Node.js 18 或更新版本
- `curl.exe`（Windows 10/11 通常已自带）

公开版直接请求游戏快照接口，**不会读取系统代理环境变量，也不包含代理地址、个人名称或个人 `user_id`**。

## 启动

双击：

```text
run-rat-drop-dashboard.bat
```

或在本目录运行：

```powershell
node rat-drop-dashboard.js
```

浏览器会打开：

```text
http://127.0.0.1:18777/
```

关闭启动窗口即可停止服务；重复双击会打开已运行的面板。

## 基本使用

1. 点击“刷新数据”，面板会抓取两次快照并分析在线/静止状态。
2. 点击“重新规划”，按右侧参数生成路线。
3. 地图支持滚轮缩放和拖动；点击团或玩家点可查看详情。
4. 点击“自定义路径”后，可以选择多个静止团，再计算经过这些团的安全路线。
5. 在地图左上角可开关路径、附加 Drop、在线玩家、贫瘠团等图层。

## 参数说明

- **体力**：单条路线的上限，`1 体力 = 10m`；建议预留体力用于战斗。
- **原点排除半径**：原点活动区范围，路线默认不会进入或穿过。
- **在线玩家避让半径**：路径绕开在线玩家的安全距离。
- **路径宽度**：计算附加 Drop 与原点附近在线威胁时使用的路线走廊宽度。
- **自己的名称 / user_id**：可选的自识别字段。填写后会自动保存，识别到自己时会以当前位置作为路线起点，并不计入自身 Drop。

## 文件说明

- `run-rat-drop-dashboard.bat`：双击启动入口。
- `rat-drop-dashboard.js`：本地 HTTP 服务、快照获取、状态分析、团聚类和路线规划。
- `rat-drop-dashboard.html`：前端界面与地图渲染。
- `rat-drop-checker.js` / `run-rat-drop-checker.bat`：早期文本版工具，仅作参考。
